Strawberries

  • The daily requirement is 70g fresh or 350g when drying.

  • The chemical yield is 2000g/m2. Note that this is using polytunnels.

  • The organic yield is 1600g/m2.

  • The land requirement is 80m2.

Blackcurrants

  • The daily requirement is 20g fresh or 100g when drying.

  • The chemical yield is 600g/m2.

  • The organic yield is 480g/m2.

  • The land requirement is 76m2.
